# mvp_dinoai

Добро пожаловать в мир приключение DinoAi!

Стек решения DinoAi: Quasar, FastAPI, ML NLTK, Hugging Face, Postgres, Celery и Redis.

Запуск DinoAi происходит очень просто.
npm run dev

После запуска вы получите
> quasar@0.0.1 dev
> quasar dev
